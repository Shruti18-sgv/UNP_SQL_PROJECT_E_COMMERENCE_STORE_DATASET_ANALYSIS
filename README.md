# UNP_SQL_PROJECT_E_COMMERENCE_STORE_DATASET_ANALYSIS

Description
This project focuses on designing and analyzing an E-Commerce store database using SQL. It includes the creation of structured tables, insertion of sample data, and execution of various queries to extract meaningful insights related to customers, products, orders, suppliers, and transactions.

The project demonstrates practical usage of SQL concepts such as joins, aggregations, transactions, and data manipulation to simulate real-world business operations in an online retail environment.
